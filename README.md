# **Cars45 Vehicle Pricing & Performance Analysis**
## **Project Overview**
This project analyzes vehicle pricing and sales performance data from Cars45 with the objective of generating data-driven insights to support pricing decisions, operational efficiency, and revenue optimization. The analysis combines structured Excel data analysis, pivot tables, and an interactive dashboard to uncover trends across car brands, pricing ranges, and transaction volumes.

The project is designed to demonstrate practical Data Analyst and Business Analyst skills, including data cleaning, exploratory analysis, KPI definition, stakeholder-focused questioning, and insight-driven reporting.
