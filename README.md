# **Cars45 Vehicle Pricing & Performance Analysis**
## **Project Overview**
This project analyzes vehicle pricing and sales performance data from Cars45 with the objective of generating data-driven insights to support pricing decisions, operational efficiency, and revenue optimization. The analysis combines structured Excel data analysis, pivot tables, and an interactive dashboard to uncover trends across car brands, pricing ranges, and transaction volumes.

The project is designed to demonstrate practical Data Analyst and Business Analyst skills, including data cleaning, exploratory analysis, KPI definition, stakeholder-focused questioning, and insight-driven reporting.

## Dataset used
- <a href="https://github.com/ugojiifechukwude1/Cars45-Visual-Dashboard">Cars45 Data

## Key Business Questions
The analysis was guided by the following questions:

1. Which car brands record the highest transaction volumes?

2. What is the average selling price by brand?

3. How are vehicles distributed across different price ranges?

4. Which brands contribute the most to total revenue?

5. Are there pricing gaps or inconsistencies across similar vehicle categories?

6. What insights can support better pricing and inventory decisions?


## Steps Taken

1. Data Understanding

   * Reviewed the dataset structure and fields.
   * Identified key variables relevant to pricing and performance analysis.
     
2. Data Cleaning & Preparation
   * Checked for missing or inconsistent values.
   * Ensured pricing fields were correctly formatted.
   * Standardized brand names for accurate aggregation

3. Exploratory Data Analysis (EDA)
   * Analyzed transaction counts by brand.
   * Examined price distributions across vehicles.
   * Identified outliers and pricing variations

4. Pivot Table Analysis
* Created pivot tables to summarize:
   * Average price by brand.
   * Total transactions per brand.
   * Revenue contribution by brand.

5. Dashboard Development
   * Built a dashboard to visualize key insights.
   * Included charts for brand performance and pricing trends.
   * Designed layout for clarity and executive-level consumption.

6. Insight Generation
   * Interpreted patterns and trends from the dashboard.
   * Connected findings to pricing and operational decisions
